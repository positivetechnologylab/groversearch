THE ALGORITHM


Grover's Search Algorithm is a powerful quantum algorithm designed to solve unstructured search problems faster than classical computers. It was developed by Lov Grover in 1996 and is one of the most well-known quantum algorithms.

Grover's search algorithm.

In classical computing, searching through an unsorted database for a specific item requires checking each item one by one, which takes linear time. On average, it will take N/2 searches to find the correct value. Grover's algorithm, on the other hand, can perform this search in roughly the square root of the number of items, resulting in a quadratic speedup. This exponential speedup becomes particularly significant for large databases.

The algorithm works by utilizing quantum principles such as superposition and interference. It creates a quantum superposition of all possible states representing the items in the database. Then, it applies a series of quantum operations to amplify the amplitude of the correct item's state and reduce the amplitudes of incorrect states. This process effectively concentrates the probability of measuring the correct state. When the algorithm is run and the final measurement is made, the correct value has a high probability.

Two important components of Grover's search algorithm are the oracle and diffuser operators:

Oracle Operator: The oracle operator is a quantum gate that marks the target solution/value to the search problem. The oracle is designed to amplify the amplitude of the target value while keeping the amplitude of other items close to zero.

Diffuser Operator: The diffuser operator helps amplify the probability amplitude of the target value while suppressing the amplitude of other incorrect solutions. It acts as an amplification step to improve the chances of finding the target value after multiple iterations of applying the oracle and diffuser. The diffuser calculates the mean amplitude of all the outcomes and then performs a phase inversion about this mean amplitude. Since the oracle operator reflected the amplitude of the target value, the target value ends up with a higher amplitude when the diffuser operator is performed.

The probability of the correct target value increases with more iterations.

In Grover's search algorithm, the oracle and diffuser operators are applied iteratively. Each iteration involves applying the oracle followed by the diffuser. This process narrows down the search space and progressively increases the probability of measuring the target value in subsequent measurements.
